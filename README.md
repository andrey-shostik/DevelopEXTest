# DevelopEx Test Task

``git clone git@github.com:denqxotl/DevelopEXTest.git``

``cd DevelopExTest/DevelopExTestBackend``

configure database.yml

``vi config/database.yml``

run

``rake db:create``

``rake db:migrate``

``rails s``

``cd ../DevelopExTestFrontend``

run

``npm install``

``grunt serve``

Open ``localhost:9000`` in your browser
